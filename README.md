# Proyecto Integrador Volt

Proyecto perro mecánico construido con Arduino.

## Objetivo
Analizar los datos generados por Volt mediante modelos de regresión y clasificación.

## Herramientas utilizadas
- Python
- Pandas
- Scikit-Learn
- NumPy

## Modelos implementados
- Regresión Lineal
- Árbol de Decisión
